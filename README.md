# 🧠 Memory Card Game

An interactive, browser-based memory card game built to challenge and improve short-term visual retention. Flip cards, find matching pairs, and try to clear the board!

[🔗 Play the Live Demo Here](https://memory-card-game-beta-sand.vercel.app/)

---

## 🚀 Features

*   **Responsive Grid:** Fully optimized layout that scales beautifully across mobile devices, tablets, and desktops.
*   **Smooth Animations:** Fluid 3D card-flip transitions and interactive hover states for an engaging user experience.
*   **State Management:** Accurately tracks flipped cards, successfully matched pairs, and prevents clicking during active matching checks.
*   **Instant Reset:** A simple restart mechanism to reshuffle the deck and start a fresh session at any time.

---

## 🛠️ Tech Stack

*   **Structure:** HTML5
*   **Styling:** CSS3 (Flexbox/Grid, 3D Transforms)
*   **Logic:** JavaScript (ES6+) — DOM manipulation, game-state logic, and card shuffling.
*   **Deployment:** Vercel

---

## 🕹️ How to Play

1. **Open the Game:** Visit the [Live Demo](https://memory-card-game-beta-sand.vercel.app/).
2. **Flip a Card:** Click on any face-down card to reveal its symbol.
3. **Find the Match:** Click a second card. 
   * If the symbols **match**, they stay face-up permanently.
   * If they **mismatch**, both cards will automatically flip back over after a brief delay.
4. **Victory:** Successfully match all pairs on the board to win!
